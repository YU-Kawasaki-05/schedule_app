# Acceptance Criteria

## 管理者ログイン

- 正しいメールアドレスとパスワードでログインできる。
- 間違った認証情報ではログインできない。
- 未ログインで管理画面を開くと `/admin/login` に誘導される。

## イベント作成

- タイトルと1件以上の候補日時でイベントを作成できる。
- タイトル未入力では作成できない。
- 候補日時が0件の場合は作成できない。
- 作成後に招待URLが表示される。
- 招待URLをコピーできる。

## 回答

- 回答者はログインなしで回答できる。
- 名前が空の場合は送信できない。
- 候補日時カードをタップして `available` / `unavailable` を切り替えられる。
- 送信後に回答完了画面が表示される。
- 同じ端末から再訪問すると回答を修正できる。

## 管理者の回答確認

- 管理者は回答者一覧を見られる。
- 候補日時ごとの参加可能人数を見られる。
- 最も参加可能人数が多い候補が分かる。
- 回答者ごとの○×が分かる。

## 締切

- 管理者はイベントを締め切れる。
- 締切後は新規回答できない。
- 締切後は回答編集できない。
- 締切後の公開回答画面には締切済み表示が出る。
