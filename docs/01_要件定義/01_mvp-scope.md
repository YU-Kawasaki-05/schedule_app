# MVP Scope

## 目的

主催者が候補日時を作成し、参加者がログインなしで行ける日を回答できる予定調整アプリを作る。

## 対象ユーザー

| ユーザー | ログイン | 主な操作 |
| --- | --- | --- |
| 管理者 | 必要 | イベント作成、編集、削除、締切、回答確認 |
| 回答者 | 不要 | 公開URLを開く、名前入力、候補選択、回答、同じ端末から修正 |

## MVPで作る範囲

- 管理者メール・パスワードログイン
- イベント作成、編集、削除、締切
- 候補日時の追加、削除、並び替え
- 招待URL発行とコピー
- 公開回答画面
- 回答完了画面
- 同じ端末からの回答修正
- 回答者別の回答一覧
- 候補日時別の参加可能人数ランキング

## MVPで作らない範囲

- 回答者ログイン
- Googleカレンダー連携
- メール通知、LINE通知
- 複数管理者、チーム機能
- 決済、公開イベント検索
- CSVエクスポート
- `maybe` 回答のUI有効化

## 技術前提

- Next.js App Router
- Supabase Auth
- Supabase Postgres
- Supabase RLS
- Tailwind CSS
- Server Actions / Route Handlers によるサーバー側検証
