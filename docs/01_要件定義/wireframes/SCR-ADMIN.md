# SCR-ADMIN Wireframes

## SCR-ADMIN-LOGIN

- メールアドレス入力
- パスワード入力
- ログインボタン
- 認証エラー表示

## SCR-ADMIN-EVENTS

- 作成済みイベント一覧
- 新規作成ボタン
- 回答数、候補数、ステータス、作成日
- 詳細、編集、削除、URLコピー

## SCR-ADMIN-EVENT-NEW / SCR-ADMIN-EVENT-EDIT

- タイトル
- 詳細
- 締切日時
- 回答結果の公開設定
- 管理者メモ
- 候補日時追加UI
- 追加済み候補一覧
- 作成または更新ボタン

## SCR-ADMIN-EVENT-DETAIL

- イベントタイトルと詳細
- 招待URLとコピーボタン
- 回答数
- 候補日時別ランキング
- 回答者別マトリクス
- 編集、締切、削除ボタン
