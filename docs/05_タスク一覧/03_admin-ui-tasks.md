# Admin UI Tasks

## SCR-ADMIN-LOGIN

- [ ] ログインフォームを作る。
- [ ] 認証エラーを表示する。
- [ ] 未設定環境変数の案内を出す。

## SCR-ADMIN-EVENTS

- [ ] イベント一覧を作る。
- [ ] 回答数、候補数、ステータス、作成日を表示する。
- [ ] 新規作成導線を作る。
- [ ] URLコピー操作を作る。

## SCR-ADMIN-EVENT-NEW / SCR-ADMIN-EVENT-EDIT

- [ ] イベントフォームを作る。
- [ ] 候補日時追加UIを作る。
- [ ] 候補削除を作る。
- [ ] 候補並び替えを作る。
- [ ] バリデーションエラーを表示する。

## SCR-ADMIN-EVENT-DETAIL

- [ ] 招待URL表示を作る。
- [ ] 候補日時別ランキングを作る。
- [ ] 回答者別マトリクスを作る。
- [ ] 締切ボタンを作る。
- [ ] 削除ボタンを作る。
