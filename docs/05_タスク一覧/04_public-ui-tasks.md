# Public UI Tasks

## SCR-PUBLIC-ANSWER

- [ ] 公開イベント表示を作る。
- [ ] 名前入力を作る。
- [ ] 候補日時カードを作る。
- [ ] `available` / `unavailable` 切り替えを作る。
- [ ] 選択数表示を作る。
- [ ] 回答送信を作る。
- [ ] localStorage から回答済み状態を復元する。

## SCR-PUBLIC-THANKS

- [ ] 回答完了画面を作る。
- [ ] 回答修正導線を作る。
- [ ] イベントページへ戻る導線を作る。

## SCR-PUBLIC-EDIT

- [ ] 編集トークン付きURLから回答を復元する。
- [ ] 無効トークンエラーを表示する。
- [ ] 締切済みエラーを表示する。

## Error States

- [ ] 存在しないイベントの表示を作る。
- [ ] 締切済みイベントの表示を作る。
- [ ] 通信エラー表示を作る。
- [ ] 候補日時変更時の案内を作る。
