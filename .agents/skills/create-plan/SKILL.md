# Create Plan

## Goal
Turn a task request into an actionable implementation plan without making changes.

## Workflow
1. Collect minimal context from relevant files.
2. Ask clarifying questions only when truly blocking.
3. Define scope and non-scope explicitly.
4. Draft ordered action items from discovery to verification.
5. Include risk and validation steps.

## Output
Return:
- Intent summary
- Scope (in/out)
- Ordered checklist
- Open questions (if any)
